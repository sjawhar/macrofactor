# 05 — Errors, Fixes, and Programming to Bench More (SBS Synthesis)

## Diagnostic-first model

SBS approach: identify where the rep fails, then match interventions.

### A) Fail at chest / cannot break bar off chest

Likely contributors:

- Front-delt limitation
- Touching too low for your structure
- Setup instability / poor control
- Less commonly, pec/triceps contribution mismatch

Primary fixes:

- Improve touch point (not excessively low)
- Rebuild setup tightness and controlled descent
- Add shoulder-focused pressing/accessories while retaining bench specificity

### B) Fail in midrange (most common)

Likely contributors:

- Straight-up initial path (technical)
- General pec + triceps strength ceiling (physical)

Primary fixes:

- Rehearse up-and-back initiation
- Increase productive bench and accessory volume over time

### C) Fail near lockout

Likely contributors:

- Elbows not rotating/flaring to strong finish position
- Triceps/pec lockout strength insufficiency

Primary fixes:

- Clean up elbow orientation on finish
- Add lockout-biased pressing/accessories while keeping full-ROM work

## Common error → fix table

| Error                                    | Performance consequence                 | SBS-aligned correction                             |
| ---------------------------------------- | --------------------------------------- | -------------------------------------------------- |
| Bar path straight up first               | Midrange sticking                       | Cue up-and-back from chest, then vertical          |
| Over-tuck to super-low touch             | Hard chest launch, triceps overload     | Touch slightly higher; moderate tuck               |
| Flaring throughout whole rep             | Shoulder irritation risk, weaker bottom | Tuck on descent, flare during ascent               |
| Inconsistent touch point                 | Variable groove and missed reps         | Set and monitor fixed touch zone                   |
| Fast uncontrolled descent                | Tendon/shoulder stress, misgroove       | 2–3s controlled eccentric                          |
| No leg-drive timing                      | Poor chest launch and path              | Initiate press with heels/glutes/chest-back intent |
| High-volume pressing with little pulling | Joint irritation accumulation           | Maintain at least 1:1 pull:press sets              |
| Ignoring wrist stack                     | Wrist/elbow discomfort                  | Low-palm bar position, wraps as needed             |

## Programming principles from SBS framing

1. **Technique is a force multiplier, not a substitute for muscle.**
2. **If technique is reasonable and progress stalls, add productive volume and muscle mass.**
3. **Use variation strategically, not randomly.**
4. **Train with maximal concentric intent on working reps.**

## Muscle Anatomy Note (Biarticularity)

- The **long head of the triceps** is biarticular — it crosses both the shoulder and elbow.
- The pecs are **not** biarticular — they cross only the shoulder.
- Shoulder position can therefore change how much long-head triceps contributes during pressing.
- At heavier loads, EMG of long-head triceps can underestimate contribution due to measurement artifacts. This is a key
  reason the pec-vs-triceps prime-mover argument is partly a measurement-method issue.

## Practical programming template logic

### For 1RM-focused phases

- High proportion of comp-style bench (grip + pause specificity).
- Keep 1–2 targeted accessories for identified weak links.
- Progress load carefully while preserving technical model.

### For growth/rebuild phases

- Keep flat bench as anchor.
- Add moderate-grip or angle variants to increase ROM stimulus.
- Build total upper-body pressing/pulling volume progressively.

## Accessory emphasis by miss location

| Miss location | Primary emphasis                   | Secondary emphasis       |
| ------------- | ---------------------------------- | ------------------------ |
| Off chest     | Front delts + path quality + setup | Pecs/triceps support     |
| Midrange      | Pecs + triceps total strength      | Bar path efficiency      |
| Lockout       | Triceps + elbow finish mechanics   | Pecs at shortened ranges |

## Progress-check metrics

- Touch-point consistency across sets.
- Fewer technical breakdowns under fatigue.
- Sticking-point bar speed improvement.
- Stable shoulder/elbow comfort week-to-week.

## Tradeoff reminder

If a change adds 2–3% load but repeatedly inflames tissue, it is rarely a winning long-term strategy.

Prefer the technique you can train hard, recover from, and repeat for months.
