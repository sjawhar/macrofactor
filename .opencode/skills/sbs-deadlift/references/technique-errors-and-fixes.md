# Deadlift Technique Errors and Fixes (SBS synthesis)

## 1) Hips shoot up immediately

### Typical causes

- Bar yanked before full tension is established.
- Hips start too low for your geometry and self-correct violently.
- Load exceeds positional strength.
- Relative quad/glute weakness vs hamstring strategy (common in sumo midrange misses).

### Immediate fixes

- Build tension gradually before break-off.
- Use “drive the floor away” for initial inches.
- Drop load to the heaviest weight that preserves shape.
- Add 1" paused deadlifts to force positional control.

## 2) Back rounds early at submax loads

### Typical causes

- Mobility limitations to reach bar with extension.
- Motor control deficits in bracing/extension.
- Setup flaws (bar too far forward, no tension build).
- True spinal erector weakness.

### Fix sequence

1. Progressive ROM pulls (blocks/rack), gradually lower start height.
2. Slow-eccentric paused RDLs for ROM + control.
3. Improve setup (bar-midfoot, lats engaged, tension-first).
4. Add erector-focused accessories if weakness remains.

## 3) Bar drifts forward / balance lost

### Typical causes

- Start position too far from midfoot.
- Knees/shins blocking path too long.
- Poor lat engagement.

### Fixes

- Re-center start over midfoot.
- Ensure knees clear path after break-off.
- Cue lats: “bar to shins,” “elbows back,” “scaps down.”
- Use pause checkpoints (off floor, below knee).

## 4) Lockout stall (hips won’t finish)

### Typical causes

- End-range hip extensor weakness.
- Weight too far forward at knee clearance.
- Sumo stance too wide for your hip anatomy.
- Over-arching instead of glute-driven finish.

### Fixes

- Hip thrusts / lockout-targeted accessories.
- Reinforce glute squeeze to neutral finish.
- Adjust sumo stance width/toe angle.
- Improve bar path and balance entering lockout.

## 5) Cannot re-extend upper back at lockout

### Typical causes

- Thoracic erector weakness under high load.

### Fixes

- Front squats.
- Above-knee block/rack pulls with strict deadlift posture.
- Rows/back extensions as adjuncts.

## 6) Grip fails first

### Typical causes

- Support grip bottleneck.
- No chalk / poor bar-in-hand placement.

### Fixes

- Chalk first.
- Mixed/hook technique cleanup.
- Timed support holds with normal-diameter bar.
- Straps for non-specific high-volume work when grip shouldn’t bottleneck target tissues.

## 7) Hyperextended lockout

### Why it matters

- Unnecessary for standards and training effect.
- Can prompt knee re-bend in competition.

### Fix

- Lockout = upright neutral finish, not exaggerated lean-back.
- Cue “glutes to finish” rather than “lean back harder.”

## 8) Reset vs touch-and-go confusion

### SBS decision rule

- If technical errors occur on first reps/heavy singles → reset between reps.
- If form stays stable and safer touch-and-go → touch-and-go is acceptable.
- Pick the style that preserves spinal/bracing quality across reps.

## 9) Over-cueing problem

- Most lifters can only execute 1–2 cues well at once.
- Prioritize the cue that addresses your highest-cost error first.
- Typical hierarchy:
  1. Maintain chest/extension intent.
  2. Initiation cue (“drive floor away”).
  3. Lockout cue (“shoulders back, hips forward”).

## 10) Troubleshooting order (high to low yield)

1. Setup geometry and balance.
2. Tension generation before break-off.
3. Cue clarity by phase.
4. Load selection that preserves positions.
5. Accessory selection for true tissue bottlenecks.
