# 04 — Common Problems and Fixes

## 1) Knee Cave

### Acceptable vs excessive

- Small “tick” inward can be common.
- Excessive valgus (especially with pain) should be corrected.

### Fix sequence

1. Narrow stance slightly.
2. Add tactile cue with mini-band around knees.
3. Strengthen abductors in squat-like patterns (e.g., hip thrust + band).

## 2) Butt Wink

### First distinction

- True lumbar flexion vs moving from overarched to neutral.

### If true butt wink

- Determine whether anatomy limits depth or control is the issue.
- Use depth-limited pin squats with progressive ROM.
- Technique progression: goblet -> front -> high bar -> low bar (if desired).

## 3) Good Morning Squat

### Pattern

- Hips shoot up, knees back, torso collapses forward.

### Immediate cues

- “Traps back into bar.”
- “Hips under bar.”

### Structural fixes

- Build quads directly; front squats are especially useful.

## 4) Forward Balance Loss

### Common contributors

- Foot pressure drift to toes, torso cave, unstable setup.

### Fixes

- Restore tripod foot pressure.
- Rehearse consistent walkout and brace sequence.
- Use solid-sole footwear.

## 5) Shoulder/Elbow/Wrist Discomfort

### Typical causes

- Grip too narrow for mobility,
- poor shelf,
- low-bar demands exceeding current mobility.

### Fixes

- Slightly wider grip, neutral wrists, thumbless grip.
- Improve shoulder external rotation + thoracic extension.
- Temporarily use high bar/front squat if symptoms persist.

## 6) “Tight Ankles” That May Not Be Mobility

- Pass a dorsiflexion wall test first.
- If ROM is adequate but knees still won’t travel, suspect strength/control limits:
  - quads,
  - calves,
  - or both.
- Add targeted training before assuming mobility-only issue.
