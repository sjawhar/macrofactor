# Progressive Overload and Phase Progression (SBS Synthesis)

## Core idea

Overload is not just "add weight". It is a phased increase in training demand that matches your current limiting factor.

- Beginner limiter: movement skill and consistency
- Intermediate limiter: muscle gain and recoverable workload
- Advanced limiter: heavy-load specificity and fatigue control

## Practical overload levers

Increase one primary lever at a time:

1. **Load** (small jumps)
2. **Reps** (within a target range)
3. **Sets** (volume progression)
4. **Exercise demand** (ROM, pause, variant complexity)
5. **Density** (same work in less time)

If multiple levers climb at once, recovery breaks first.

## Methods

### 1) Linear progression (best for beginners)

- Keep exercise menu stable
- Add load session to session
- Keep 3+ RIR and technical consistency
- Exit when weekly increases require grinding

Example:

- Squat 3x5 @ 100 kg → 102.5 → 105 ...
- Stop adding once reps become technical maxes rather than training reps

### 2) Double progression (default for accessories)

- Choose rep range (e.g., 8-12)
- Progress reps at fixed load until top of range is achieved across all sets
- Add load and restart near lower rep bound

Example:

- Week 1: 27.5 kg x 10/9/8
- Week 2: 27.5 kg x 11/10/9
- Week 3: 27.5 kg x 12/11/10
- Week 4: 27.5 kg x 12/12/12
- Week 5: 30 kg x 9/8/8

### 3) Wave/block progression (intermediate and advanced)

Use 3-8 week waves with rising specificity/intensity or volume concentration.

Example (strength-focused wave):

- W1: 5x5 @ 75%
- W2: 4x4 @ 80%
- W3: 5x3 @ 85%
- W4: Deload/pivot

## Phase progression timeline (SBS-style)

### Beginner phase (~2-6 months typical)

- Goal: fast neural/technical gains
- 2-4 exposures per lift weekly
- Loads typically 60-80%
- Stay far from failure
- Progress frequently (session-to-session)

### Intermediate phase (~3-8 years often)

- Goal: add muscle and expand work capacity
- Main lifts 75-85% most of the time
- Bulk of volume from accessory lifts (6-15 reps)
- PR cadence evolves: ~4-week blocks → 8-week → 12-week

### Advanced phase (open-ended)

- Goal: convert existing muscle into max-specific performance
- Offseason: high variation + work capacity for main lifts
- Pre-meet: reduce volume/frequency, increase intensity/specificity
- Compete ~2x/year to practice execution skills

## Progression checkpoints

Use this decision ladder weekly:

1. Did performance trend up at matched effort? If yes, continue.
2. If no: are sleep/stress/nutrition limiting? Fix first.
3. If recovery is poor: reduce volume 20-40% for 1 week.
4. If recovered but stagnant: change progression model or exercise variant.
5. If still stalled: reassess phase (you may have outgrown current model).

## Red flags

- Grinding most work sets
- PR attempts too frequent for training age
- Main-lift volume too high to recover from
- Running beginner progression after repeated stalls
