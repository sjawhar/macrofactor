# Volume Landmarks and Weekly Set Targets

## Why volume matters in SBS programming

SBS emphasizes that **training volume is the primary driver of hypertrophy** once basic proficiency is in place. More muscle increases long-term strength potential.

From the source guide's implementation section:

- Train major muscles **2-3x/week**
- Perform **4-6 sets per session**
- Practical baseline = **8-18 sets/week**

This document extends that baseline into MEV/MAV/MRV-style planning.

## Landmark definitions

- **MEV:** minimum weekly sets that produce measurable progress
- **MAV:** range where gains are best for most people
- **MRV:** highest weekly sets recoverable without performance decline

Landmarks move with sleep, stress, exercise selection, and training age.

## Weekly direct set landmarks

Hard sets only (roughly RPE 6-9 / 4-1 RIR).

| Muscle     |   MEV |   MAV |   MRV |
| ---------- | ----: | ----: | ----: |
| Chest      |  8-10 | 12-18 | 20-24 |
| Back       | 10-12 | 14-22 | 24-28 |
| Quads      |  8-10 | 12-18 | 20-24 |
| Hamstrings |   6-8 | 10-16 | 18-22 |
| Glutes     |   6-8 | 10-18 | 20-24 |
| Delts      |  8-10 | 12-20 | 22-26 |
| Triceps    |   6-8 | 10-16 | 18-22 |
| Biceps     |   6-8 | 10-16 | 18-22 |
| Calves     |   6-8 | 10-16 | 18-22 |
| Abs        |   4-6 |  8-12 | 14-18 |

## Set counting and overlap

Use pragmatic overlap accounting:

- Compound pressing contributes to chest/delts/triceps
- Rows/pull-ups contribute to lats/upper back/biceps
- Squat/lunge patterns contribute to quads + glutes
- Hinge patterns contribute to glutes + hamstrings + erectors

Avoid double-counting every compound set as full credit to every involved muscle. A simple rule:

- Prime mover = full set credit
- Strong secondary = ~0.5 set credit

## How to find YOUR landmarks

Start near low-MAV for 2-4 weeks, then adjust:

- If performance and recovery both improve: add 2-4 sets/week for target muscle
- If progress flat but recovery good: add 2 sets/week
- If soreness/fatigue and performance worsen: reduce 20-40% for 1 week

When repeated overreach appears at similar volume, that level is near MRV.

## Session distribution rules

- Keep per-session sets manageable (often 3-8 hard sets per muscle)
- Spread weekly volume across 2-4 sessions for better quality
- Use accessories to carry high-volume work with lower joint stress

## Example (intermediate chest plan)

Week 1-2: 12 sets/week (MEV-to-low MAV)

- Day 1 bench: 4 sets
- Day 2 incline DB: 4 sets
- Day 4 machine press/fly: 4 sets

Week 3-4: 14-16 sets/week if recovery is strong

Week 5: deload to ~8 sets if fatigue accumulates

## Common errors

- Chasing MRV every week
- Ignoring overlap from compounds
- Keeping volume constant despite clear adaptation changes
- Using failure training to compensate for poor volume planning
